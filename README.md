# 旅行日志列表组件（React）
这是一个基于 React 开发的旅行日志展示组件，通过数据驱动渲染旅行记录列表，支持展示旅行图片、目的地、日期、详情及谷歌地图跳转功能。

## 功能说明
- 渲染旅行日志列表：自动遍历数据源，生成结构化的日志卡片
- 展示核心信息：包含旅行图片、目的地国家、谷歌地图链接、旅行日期、日志正文
- 响应式布局：通过 CSS 类名（`container`、`journal-entry` 等）适配不同屏幕
- 语义化结构：使用 `article`、`main` 等标签优化页面语义

## 文件结构
```
src/
├── components/
│   ├── Header.js       # 页面头部组件（如标题、导航）
│   └── Entry.js        # 单个旅行日志卡片组件
├── pages/
│   └── App.js          # 主应用组件（数据处理与列表渲染）
└── data.js             # 旅行日志数据源（包含日志图片、地点、日期等信息）
```

## 安装与运行
### 前提条件
- 已安装 Node.js（v14+ 推荐）
- 熟悉 React 基础环境（基于 Create React App 或 Vite 搭建）

### 步骤
1. 克隆或下载项目代码到本地
2. 进入项目根目录，安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm start
   ```
4. 打开浏览器访问 `http://localhost:3000` 即可查看效果

## 核心代码解析
### 1. 主应用组件（App.js）
- 导入依赖：Header 组件（头部）、Entry 组件（日志卡片）、data 数据源
- 数据处理：通过 `data.map()` 遍历数据源，生成日志卡片数组 `entryElements`
- 传参方式：使用对象扩展运算符（`{...entry}`）批量传递日志数据到 Entry 组件
- 列表渲染：将生成的日志卡片数组嵌入页面主容器

### 2. 日志卡片组件（Entry.js）
- 接收 props：包含旅行日志的完整数据（img、country、googleMapsLink 等）
- UI 渲染：
  - 主图片：通过 `props.img.src` 和 `props.img.alt` 绑定图片资源与描述
  - 地点信息：展示国家名称，绑定谷歌地图跳转链接
  - 核心内容：渲染旅行标题、日期、正文文本
- 样式结构：通过预定义 CSS 类名控制布局和外观（需配合对应的 CSS 文件使用）
